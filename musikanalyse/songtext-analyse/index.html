Songtext-Analyse

Wie oft kommt ein Wort in Songtexten vor? Gib ein Suchwort ein, wähle einen Künstler oder ein Album – die Auswertung erscheint als Säulendiagramm. Daten von genius.com.

Hinweis: Texte werden live von genius.com geladen – das kann einige Sekunden dauern; sehr umfangreiche Künstler werden auf die bekanntesten Lieder begrenzt. Damit das zuverlässig klappt, am besten unten die eigene Vermittler-Adresse eintragen.

Eigene Vermittler-Adresse (empfohlen)

Trage hier die Adresse deines eigenen Vermittlers (Cloudflare-Worker) ein, z. B. https://songtext-proxy.deinname.workers.dev. Sie wird nur in deinem Browser gespeichert. Ist das Feld leer, werden öffentliche Dienste versucht (oft unzuverlässig).
